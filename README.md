## Installation

1. Clone repository

   ```
   git clone https://github.com/chrisnph/node-ts-express-boilerplate
   ```

2. Install packages

   ```
   cd node-ts-express-boilerplate && npm i
   ```

3. Run Project Locally

   ```
   npm run start:local
   ```

   or if you have nodemon installed globally

   ```
   npm run start
   ```
